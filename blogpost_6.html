GSoC 2022 has finished for Xfce - a brief summary

We are happy to announce that all our Xfce GSoC participants, namely Yogesh Kaushik, Amrit Borah and Pratyaksh Gautam all successfully finished their Google Summer of Code 2022.

They brought a lot of nice new features to Xfce. Here is a short sum up on the projects they worked on:

Yogesh Kaushik

  • Thunar Bulk Renamer - Gracefully handle name clashes

    E.g. When the new name for one file matches the old name of the second file. Such will work fine now.

  • Xfce4-Screenshooter - Custom Actions

    Provide the possibility to add custom actions that can be executed once the screenshot is taken.

  • Xfce4-panel - Merge DateTime with Clock plugin

    DateTime and Clock plugin plug-in were somewhat overlapping. Now, all the features of these plug-ins are merged into the Clock plug-in.

Amrit Borah

  • Thunar - Support for file highlighting

    A feature to highlight files/folders with different colors

  • Xfce4-Terminal - User Profiles

    Enables users to create multiple different profiles and switch between them. The feature is work in progress.

  • Thunar - Tree-View as part of the list-view

    Expandable list-view folders. The feature is work in progress.

Pratyaksh Gautam

  • Thunar - Support undo/redo for file operations

    You did something stupid ? Not a problem anymore! Now you can undo/redo the most recent operations.

  • Thunar - Count of containing files in size column of directories

    In list view, the size column now optionally shows the number of containing files for each folder.

  • Thunar - Introduce a gallery view

    View optimized to show only pictures in a gallery. Filenames will only be shown on hover. The feature is work in progress.

While some of the features are not yet merged, most of them already got finished in time for Xfce 4.18 which is planned to be released on 15th Dec.

I hope you will enjoy !
